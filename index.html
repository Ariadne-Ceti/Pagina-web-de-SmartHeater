SmartHeater

La nueva generación de calentadores inteligentes, eficientes y ecológicos.

Objetivo General

Desarrollar un boiler inteligente controlado por internet, capaz de calentar agua de forma segura y eficiente,
monitorear su estado y enviar alertas al usuario.

Objetivos Específicos

Integrar sensores de temperatura, nivel y corriente para un control preciso.
Automatizar el encendido y apagado con control remoto.
Diseñar una aplicación que gestione tu boiler en tiempo real recibiendo alertas de seguridad y mantenimiento.
Crear un diseño ecológico, con componentes reemplazables y reutilizables.

Nuestro Prototipo

Nuestro prototipo tiene un tanque de 20 litros, ideal para 1 o 2 personas, y alcanza una temperatura máxima de 65 a 70 °C en aproximadamente 35 a 40 minutos, gracias a una resistencia eléctrica de 1500 W.

Planteamiento del Problema

Los boilers tradicionales consumen energía o gas durante todo el día, incluso cuando no se utilizan, lo que genera desperdicio energético y emisiones contaminantes.

Además, cuando un boiler se daña, generalmente se desecha completo, aunque solo una pieza esté defectuosa, contribuyendo a la basura electrónica.

Frente a esta problemática, SmartHeater surge como un prototipo inteligente y modular que:

Solo calienta el agua cuando se necesita.

Permite reemplazar componentes fácilmente.

Y ofrece monitoreo remoto y alertas automáticas para mayor seguridad y eficiencia.

Tecnología y Funcionamiento Interno


El prototipo SmartHeater combina hardware, software y comunicación IoT para simular el funcionamiento de un boiler inteligente.

ESP32: Microcontrolador principal, con conexión Wi-Fi; controla sensores, relés y comunicación con la nube.

Sensor de Temperatura: : Mide la temperatura del agua y apaga el sistema al alcanzar el valor programado.


Rele: : Actúan como interruptores electrónicos para el control de la resistencia.


MCB (Mini Circuit Breaker): Protege el circuito ante cortos o sobrecargas, apagando el sistema automáticamente.

Firebase (Base de datos en la nube): Conecta la app móvil con el ESP32, enviando comandos y recibiendo datos en tiempo real.

Aplicación Móvil

La aplicación móvil de SmartHeater permite controlar el sistema de calefacción de manera remota desde cualquier lugar, brindando comodidad, seguridad y eficiencia energética.

A través de una interfaz intuitiva, el usuario puede encender o apagar el boiler, monitorear la temperatura en tiempo real y recibir alertas ante cualquier anomalía en el sistema.

Además, la aplicación almacena datos históricos que permiten analizar el consumo energético, ayudando al usuario a optimizar el uso del calentador.

  • Control remoto desde smartphone
  • Monitoreo en tiempo real
  • Alertas de seguridad automáticas
  • Historial de uso y consumo

Datos Técnicos del Prototipo

  • Capacidad del tanque: 40 litros
  • Potencia de resistencia: 2000 W
  • Voltaje de operación: 127 V
  • Temperatura máxima: 70 °C
  • Tiempo de calentamiento: 35 - 40 minutos
  • Microcontrolador: ESP32 con Wi-Fi integrado
  • Sensor de temperatura tipo NTC
  • Sistema de protección con MCB

Eficiencia y Enfoque Ecológico

SmartHeater está diseñado para reducir significativamente el consumo energético en comparación con los boilers tradicionales, ya que solo opera cuando es necesario.

Gracias a su sistema inteligente, evita el desperdicio de energía al mantener el agua caliente únicamente cuando el usuario lo requiere, lo que se traduce en ahorro económico y menor impacto ambiental.

Reduce el consumo de energía eléctrica.
Disminuye emisiones contaminantes.
Evita el sobrecalentamiento innecesario.
Promueve el uso responsable de recursos.

¿Cómo Surge la Idea?

La idea de SmartHeater surge a partir de la problemática observada en los hogares, donde los boilers tradicionales consumen energía de forma constante, incluso cuando no están en uso.

Además, se identificó que muchos sistemas carecen de monitoreo y control, lo que puede generar riesgos como sobrecalentamiento o fallas eléctricas.

A partir de estas necesidades, se planteó el desarrollo de un sistema inteligente capaz de optimizar el consumo energético, mejorar la seguridad y brindar control remoto al usuario.
